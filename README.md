# ED_Lista02.02
## Divisão: Subtração Recursiva

Este desafio consiste em criar uma aplicação, em Java, que receba dois números naturais e apresente o resto da divisão entre eles.

```
Requisitos do projeto:

1. Arquitetura MVC;
2. A divisão deve ser realizada por subtrações em uma função recursiva;
3. Declarar, via comentário, a condição de parada da função.
```
